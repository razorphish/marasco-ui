Funding Miracles

FREE online fundraising

Funding Miracles.com, project is a FREE online fundraising designed to not only help those in need but significantly reduce the cost for the donors

A web application as well as a backend data manager was produced to manage the data going in and out of the application.

The application currently uses the following platforms: Angular 1.5, Bootstrap and .NET Web Api on an Azure Cloud based network and the project took about 8 months to complete. Currently, Funding Miracles is the leading FREE fundraising platform

Data Manager 2.0

Malldub

This is another cool project

The Malldub project was a do-it-yourself mall building application designed to make it extremely easy to build a "mall" complete with different "stores" and ma-services offered to the users.

In addition, the service was offered at a low-cost introductory rate with a few monetizing services (ad sharing and store rentals) included with every mall.

The application currently uses the following platforms: Angular 4, Bootstrap and NODEJS and the project took about 4 months to complete.
